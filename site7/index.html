Exemplo: HTML + CSS com <pre> (pré-código)

Dois arquivos: index.html e styles.css

Pré-código HTML

Use o elemento <pre> para exibir código preservando quebras e espaços:

<!-- Exemplo: Um simples botão HTML -->
<button class="btn-primary">Clique aqui</button>

Pré-código CSS

Também é comum mostrar snippets de CSS:

/* Exemplo CSS */
.btn-primary {
padding: 0.6rem 1rem;
border-radius: 6px;
background: linear-gradient(135deg,#4f46e5,#06b6d4);
color: white;
border: none;
}
